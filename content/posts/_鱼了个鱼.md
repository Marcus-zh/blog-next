## 经过

#### 1.首先clone下来

```shell
git clone https://github.com/liyupi/yulegeyu.git
```

#### 2.打开VS code

#### 3.下载依赖

```shell
npm install
```

不过我下载的有点慢,不知道是不是我的问题

#### 4.生成静态文件

```shell
npm run build
```

#### 5.运行

```shell
npm run dev
```
